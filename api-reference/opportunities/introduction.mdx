---
title: "Oportunidades"
description: "Api para cadastro de oportunidades externas dentro da Fretatech."
---

O novo módulo de **Oportunidades** permite criar e gerenciar registros de oportunidades externas, ou seja, você pode enviar suas oportunidades de uma LandingPage, do Whatsapp ou qualquer outro sistema externo.\
Com suporte a **campos dinâmicos (repeater)** definidos via painel ou API, As oportunidades são exibidas no **funil de vendas**, podem ser criadas/atualizadas via endpoint autenticado e podem ser convertidas em **orçamentos**.

<Info>
  Para utilizar o módulo de oportunidades entre em contato com o suporte e solicite sua chave de acesso.
</Info>
