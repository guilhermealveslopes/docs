---
title: "Busca de Veículos"
description: "Consulte disponibilidade e preços em tempo real"
---

A busca de veículos é **assíncrona**: você inicia a busca e depois consulta os resultados por polling. Isso acontece porque o GDS consulta múltiplos fornecedores em paralelo, e cada um pode levar alguns segundos para responder.

O GDS resolve os endereços automaticamente via FretaMaps — você não precisa fornecer coordenadas, basta enviar o endereço em texto livre.

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| `POST` | `/search` | Iniciar busca de veículos |
| `GET` | `/search/{search_db_id}` | Consultar resultados (polling) |

---

## Como funciona o fluxo

```
1. POST /search  →  recebe search_db_id (status: pending)
        |
        | (aguarde 2-5 segundos)
        v
2. GET /search/{id}  →  status: pending | complete | failed
        |
        | (repita enquanto status = pending)
        v
3. Quando status = complete  →  lista de veículos disponível
```

Recomendamos polling a cada **2 segundos**, com timeout máximo de **30 segundos**.

---

## Iniciar busca

```bash
curl -X POST https://gds.fretatech.com.br/api/partner/v1/search \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "origin": "Aeroporto de Guarulhos, Guarulhos - SP",
    "destination": "Av. Paulista, 1000, São Paulo - SP",
    "trip_date": "2026/04/15",
    "trip_time": "10:00",
    "trip_type": "onewaytrip",
    "passengers": 40
  }'
```

### Campos da requisição

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `origin` | string | Sim | Endereço de origem em texto livre |
| `destination` | string | Sim | Endereço de destino em texto livre |
| `trip_date` | string | Sim | Data da viagem no formato `YYYY/MM/DD` |
| `trip_time` | string | Sim | Horário de partida no formato `HH:MM` |
| `trip_type` | string | Sim | Tipo da viagem (veja tabela abaixo) |
| `passengers` | integer | Não | Número de passageiros |

### Tipos de viagem

| Valor | Descrição |
|-------|-----------|
| `onewaytrip` | Ida simples |
| `roundtrip` | Ida e volta |
| `hourtrip` | Por hora (com franquia) |
| `customtrip` | Roteiro personalizado (múltiplos trechos) |

### Resposta

```json
{
  "success": true,
  "data": {
    "search_db_id": 123,
    "status": "pending",
    "origin": {
      "description": "Aeroporto Internacional de Guarulhos",
      "lat": -23.4356,
      "lng": -46.4731
    },
    "destination": {
      "description": "Av. Paulista, 1000 - Bela Vista, São Paulo - SP",
      "lat": -23.563,
      "lng": -46.6543
    },
    "message": "Busca iniciada. Use GET /search/{id} para consultar."
  }
}
```

A resposta inclui os endereços resolvidos pelo GDS (com coordenadas confirmadas) para você validar se o local foi interpretado corretamente.

## Consultar resultados (polling)

```bash
curl -X GET "https://gds.fretatech.com.br/api/partner/v1/search/123" \
  -H "Authorization: Bearer SUA_API_KEY"
```

### Valores de status

| Status | Descrição |
|--------|-----------|
| `pending` | Busca em andamento — continue o polling |
| `complete` | Busca concluída — resultados disponíveis |
| `failed` | Erro na busca — tente novamente |

### Resposta quando status = pending

```json
{
  "success": true,
  "data": {
    "search_db_id": 123,
    "status": "pending",
    "vehicles": []
  }
}
```

### Resposta quando status = complete

```json
{
  "success": true,
  "data": {
    "search_db_id": 123,
    "status": "complete",
    "vehicles": [
      {
        "id": "abc123",
        "name": "Ônibus Executivo 50 lugares",
        "capacity": 50,
        "type": "bus",
        "price_cents": 180000,
        "tenant_id": "tenant-uuid-xyz",
        "tenant_name": "Viação Exemplo"
      },
      {
        "id": "def456",
        "name": "Van Executiva 16 lugares",
        "capacity": 16,
        "type": "van",
        "price_cents": 85000,
        "tenant_id": "tenant-uuid-abc",
        "tenant_name": "Transporte Premium Ltda"
      }
    ]
  }
}
```

### Estrutura do veículo

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | Identificador do veículo (use para criar o orçamento) |
| `name` | string | Nome/descrição do veículo |
| `capacity` | integer | Capacidade de passageiros |
| `type` | string | Tipo do veículo (`bus`, `van`, `minibus`) |
| `price_cents` | integer | Preço em centavos (valor do fornecedor, sem margem) |
| `tenant_id` | string | Identificador do fornecedor |
| `tenant_name` | string | Nome do fornecedor |

<Note>
  O campo `price_cents` representa o valor bruto do fornecedor. A margem configurada no GDS será aplicada ao criar o orçamento. Veja [Orçamentos](/partner/quotes) para mais detalhes.
</Note>

## Exemplo de polling em JavaScript

```javascript
async function waitForSearchResults(searchDbId, apiKey) {
  const maxAttempts = 15; // 30 segundos com intervalo de 2s

  for (let i = 0; i < maxAttempts; i++) {
    const response = await fetch(
      `https://gds.fretatech.com.br/api/partner/v1/search/${searchDbId}`,
      { headers: { Authorization: `Bearer ${apiKey}` } }
    );

    const { data } = await response.json();

    if (data.status === 'complete') return data.vehicles;
    if (data.status === 'failed') throw new Error('Busca falhou. Tente novamente.');

    await new Promise(resolve => setTimeout(resolve, 2000));
  }

  throw new Error('Timeout: busca não concluiu em 30 segundos.');
}
```
