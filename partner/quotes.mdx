---
title: "Orçamentos"
description: "Crie e consulte orçamentos de fretamento"
---

Um orçamento (quote) é criado a partir dos resultados de uma busca de veículos. Ele representa uma proposta de fretamento para um cliente, com o preço do fornecedor acrescido da margem configurada.

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| `POST` | `/quotes` | Criar orçamento |
| `GET` | `/quotes` | Listar orçamentos |
| `GET` | `/quotes/{id}` | Buscar orçamento por ID |

---

## Criar orçamento

O payload é construído a partir dos dados retornados pela [busca de veículos](/partner/search). Use os campos `trip` e `vehicles` do resultado do polling.

### Campos da requisição

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `customer_id` | integer | Sim | ID do cliente no GDS |
| `simpleTrip` | object | Sim | Dados da viagem (veja abaixo) |
| `selectedVehicle` | object | Sim | Veículo escolhido nos resultados |
| `selectedVehicle.id` | integer | Sim | ID do veículo |
| `selectedVehicle.priceCents` | number | Sim | Preço do veículo (`price_cents` retornado pela busca) |
| `selectedVehicle.tenant_id` | string | Sim | UUID do fornecedor (`tenant_id` retornado pela busca) |
| `customRouter` | array | Só para `customtrip` | Array de paradas do roteiro (veja abaixo) |
| `search_id` | string | Não | ID da busca de origem (vincula orçamento à busca) |
| `default_margin_percentage` | number | Não | Margem em percentual (ex: `15` para 15%). Se omitido, usa a margem padrão configurada na conta. |

#### Objeto `simpleTrip`

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `tripType` | string | Sim | Tipo da viagem: `onewaytrip`, `roundtrip`, `customtrip`, `hourtrip` |
| `origin` | object | Sim | Local de partida (use os dados retornados pelo polling da busca) |
| `origin.address` | string | Sim | Endereço completo |
| `origin.city` | string | Sim | Cidade |
| `origin.date` | string | Sim | Data de partida no formato `DD/MM/YYYY` |
| `origin.time` | string | Sim | Horário de partida no formato `HH:MM` |
| `origin.lat` | string | Sim | Latitude |
| `origin.lng` | string | Sim | Longitude |
| `origin.address_components` | object | Sim | `state`, `state_short`, `city` |
| `destination` | object | Sim | Local de chegada (mesma estrutura de `origin`) |
| `destination.date` | string | Sim | Para `roundtrip`: data de retorno. Para `onewaytrip`: data de chegada. |

### Ida e volta (`roundtrip`)

Para ida e volta, use `tripType: "roundtrip"` e informe datas diferentes em `origin` (partida) e `destination` (retorno). O preço do veículo já contempla a viagem completa de ida e volta.

  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": 5,
    "search_id": "Ox2VB6xxOL4HOVae",
    "default_margin_percentage": 15,
    "simpleTrip": {
      "tripType": "roundtrip",
      "origin": {

## Como a margem é aplicada

O `priceCents` do veículo é o valor bruto do fornecedor (em reais). O GDS converte internamente e aplica a margem:

```
subtotal = priceCents (convertido para centavos)
service_fee = subtotal × service_fee_percentage
margin = subtotal × margin_percentage
total = subtotal + service_fee + margin
```

Exemplo com `priceCents: 41586.16` e margem de 15%:
```
subtotal:     R$ 41.586,16  (4.158.616 centavos)
service_fee:  R$  2.911,03  (7% padrão)
margin:       R$  6.237,92  (15%)
total:        R$ 50.735,11
```

## Status do orçamento

| Status | Descrição |
|--------|-----------|
| `pending` | Orçamento criado, aguardando aprovação do cliente |
| `approved` | Aprovado pelo cliente |
| `rejected` | Recusado pelo cliente |
| `converted` | Convertido em pedido |
| `expired` | Prazo de validade expirado |

---

## Listar orçamentos

```bash
curl -X GET "https://gds.fretatech.com.br/api/partner/v1/quotes?status=pending&customer_id=5" \
  -H "Authorization: Bearer SUA_API_KEY"
```

### Parâmetros de filtro

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `status` | string | Filtra por status do orçamento |
| `customer_id` | integer | Filtra por cliente |
| `date_from` | string | Data inicial (`YYYY-MM-DD`) |
| `date_to` | string | Data final (`YYYY-MM-DD`) |

---

## Buscar orçamento por ID

```bash
curl -X GET https://gds.fretatech.com.br/api/partner/v1/quotes/76 \
  -H "Authorization: Bearer SUA_API_KEY"
```

Retorna o objeto completo do orçamento, idêntico ao retorno da criação.
